Paper 3 Question 1: Give Two Differences (4 Marks)

Exam Technique Tip: Keep this response concise. For each difference, state the difference clearly, then provide a direct comparison quote/detail from Source A and Source B. Do not analyze NOP or add own knowledge.
Difference 1 Builder
One difference between the sources is that Source A shows...
Whereas Source B shows...
For example, Source A states: "___________________________________"
which differs from Source B which states: "_________________________"
Difference 2 Builder
Another difference is that Source A portrays...
Whereas Source B portrays...
For example, Source A shows: "___________________________________"
which differs from Source B which shows: "_________________________"

Paper 3 Question 2: Explain Why (12 Marks)

Exam Technique Tip: You must explain **three separate reasons** for the event. For each reason, write a PEEL paragraph (Point, Evidence, Explanation, Link). Focus directly on causation ("Explain why...").
Paragraph 1 (First Reason)
Point: One key reason why [event] occurred was...
Evidence: For example, during this time... [add specific detail]
Explanation: This led to [event] because... [explain the cause]
Link: Therefore, this was a vital factor because...
Paragraph 2 (Second Reason)
Point: Another significant reason was...
Evidence: This can be seen in...
Explanation: This caused [event] to happen because...
Link: As a result, this escalated the situation by...
Paragraph 3 (Third Reason - From Own Knowledge)
Point: Furthermore, [event] was also caused by...
Evidence: Specific historical context shows that...
Explanation: This played a role in the outcome because...
Link: Thus, this completed the path to...

Paper 3 Question 3a: Source Utility (8 Marks)

Exam Technique Tip: Evaluate **both** sources. For each source, analyze its content (what it tells you), its Nature, Origin, and Purpose (NOP - why it was made and if it's reliable), and back it up with your Own Knowledge.
Source B Evaluation Paragraph
Source B is useful because the content reveals... [extract quotes]
This is supported by my own knowledge that... [add historical context]
Furthermore, Source B's utility is enhanced/limited by its NOP because...
It was produced by [Author] in [Year] to [Purpose]...
Source C Evaluation Paragraph
Source C is also useful because it shows...
From my own knowledge, I know that...
Regarding NOP, Source C is useful because it is a [Nature] from...
which means its reliability and perspective are...
